Webeando
========

Webeando Facil es un curso para la creacion de aplicasiones web facil y rapido.

###Cronograma del curso:

- Intoduccion a la programacion web.
- Construccion de una GUI con BuildingBlocks.
- Javascrip.
- Web API's.
- Montar una App de TODO.
